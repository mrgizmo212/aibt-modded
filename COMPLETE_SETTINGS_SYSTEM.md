## üéØ Overview

A **3-tier configuration system** for AI model parameters:

1. **Global Defaults** (Admin) - System-wide settings for all users
2. **Per-Model Settings** (User) - Custom settings per trading model
3. **Runtime Overrides** (Trading Session) - Temporary overrides during trading

---

## üìã Features Implemented

### ‚úÖ **Comprehensive Parameter Support**
- **Input Tokens** - `max_prompt_tokens` (up to 128K)
- **Output Tokens** - `max_completion_tokens` / `max_tokens` / `max_output_tokens`
- **Temperature** - 0.0-2.0 (where supported)
- **Verbosity** - low/medium/high (GPT-5 only)
- **Reasoning Effort** - minimal/low/medium/high (GPT-5, o3, o3-mini)
- **Top-p** - 0.0-1.0 (nucleus sampling)
- **Top-k** - 1-500 (Claude, Gemini)
- **Frequency Penalty** - -2.0 to 2.0
- **Presence Penalty** - -2.0 to 2.0  
- **Web Search** - true/false (Grok only)

### ‚úÖ **Model-Specific Intelligence**
- **GPT-5** ‚Üí NO temperature, uses verbosity + reasoning_effort
- **o3 / o3-mini** ‚Üí NO temperature, reasoning_effort only
- **Claude** ‚Üí Full parameters + top_k
- **Gemini** ‚Üí Uses max_output_tokens instead of max_tokens
- **Grok** ‚Üí Includes web_search capability
- **Others** ‚Üí Standard temperature-based parameters

### ‚úÖ **Two-Level Configuration**
- **Global** ‚Üí Stored in `global_settings` table
- **Per-Model** ‚Üí Stored in `models.model_parameters` column

## üìù Parameter Descriptions

### **Input/Output Tokens:**
- **max_prompt_tokens** - Maximum input tokens (context window)
- **max_completion_tokens** - Maximum output tokens (response length)
- **max_tokens** - Alias for max_completion_tokens (most models)
- **max_output_tokens** - Gemini-specific output limit

### **Temperature:**
- **0.0** ‚Üí Deterministic, precise
- **0.7** ‚Üí Balanced (recommended for trading)
- **1.5** ‚Üí Creative, diverse
- **2.0** ‚Üí Maximum randomness

### **Verbosity** (GPT-5 only):
- **low** ‚Üí Concise responses
- **medium** ‚Üí Balanced detail
- **high** ‚Üí Comprehensive explanations

### **Reasoning Effort** (GPT-5, o3):
- **minimal** ‚Üí Quick decisions
- **low** ‚Üí Basic analysis
- **medium** ‚Üí Standard reasoning
- **high** ‚Üí Deep analysis (recommended for trading)

### **Top-p** (Nucleus Sampling):
- **0.1** ‚Üí Very focused
- **0.9** ‚Üí Balanced (recommended)
- **1.0** ‚Üí Full vocabulary

### **Top-k** (Vocabulary Limit):
- **1-50** ‚Üí Very restricted
- **250** ‚Üí Balanced (Claude default)
- **500** ‚Üí Maximum variety

### **Frequency Penalty:**
- **< 0** ‚Üí Allow repetition
- **0.0** ‚Üí Neutral (default)
- **> 0** ‚Üí Discourage repetition

### **Presence Penalty:**
- **< 0** ‚Üí Allow staying on topic
- **0.0** ‚Üí Neutral (default)
- **> 0** ‚Üí Encourage new topics

### **Web Search** (Grok):
- **true** ‚Üí Enable real-time web search
- **false** ‚Üí Use training data only
